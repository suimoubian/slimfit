# SlimFit — 减肥打卡助手

一个纯粹自用的减肥打卡 PWA 应用，支持电脑端和手机端，可打包为 APK 安装。

---

## 主要功能

### 打卡追踪
- **默认打卡 + 多种自定义打卡**：运动、饮食、饮水……类型随意增减
- **礼炮粒子动画**：每次打卡按钮像礼炮一样喷射彩带、星星和圆点
- **7 阶段成长体系**：从「迈出第一步」到「破茧成蝶」，每个阶段有专属鼓励语
- **阶段升级庆祝**：跨阶段时全屏粒子动画 + 大字标题
- **成就徽章**：坚持一周、月度之星、百日英雄、自律大师、钢铁意志

### 日历热力图
- 类似 GitHub 贡献图的方格日历
- 每种打卡类型独立日历，颜色区分
- 支持左右翻月，点击查看打卡详情
- 全部/各类型一键切换

### 健康计算
- BMI 计算 + 环形进度条可视化
- 基础代谢 (BMR) 估算
- 6 级体重分类 + 理想体重范围 + 针对性建议

### 减肥餐推荐
- 17 道中式减脂菜谱，覆盖早中晚三餐
- 每道菜包含：食材清单、精确用量、卡路里、分步烹饪法、营养依据
- 支持全天餐 / 单餐生成，可换一批
- 食谱支持远程更新，无需升级 App 即可获取新菜谱

### 减肥计划
- **6 档方案**：体验减 → 随心减 → 认真减 → 严格减 → 冲刺减 → 塑形减
- 热量缺口从 150kcal 到 900kcal，难度从散步到每周 6 练
- 每档包含：适用/不适用人群、风险提示、周一到周日食谱表、运动计划、预期时间线
- 3 题问卷智能推荐最合适方案

### 周期总结
- 本周 / 本月 / 本季度打卡报告
- 打卡率、打卡分布横向条形图
- 最长连续天数、当前阶段回顾
- AI 生成周期小结

### 个性化
- 8 种预设柔和主题色 + 自定义取色器
- 纯色 / 渐变 / 图片背景
- 暗黑模式、字号调节
- 数据 JSON 导入导出

---

## 技术栈

纯 HTML + CSS + JavaScript 单页应用，零框架依赖。

- **LocalStorage** 本地数据持久化
- **CSS Custom Properties** 动态主题系统
- **Canvas API** 粒子动画引擎
- **PWA** 可安装到桌面 / 主屏幕
- **Capacitor** 打包为 Android APK

## 如何使用

### 方式一：浏览器打开
直接在浏览器打开 `index.html` 即可使用全部功能。

### 方式二：安装 PWA
将文件夹部署到任意静态服务器，手机浏览器打开后「添加到主屏幕」。

### 方式三：安装 APK
从 [Releases](../../releases) 下载最新 APK，Android 手机直接安装。

---

## 自行构建 APK

本仓库包含 GitHub Actions 工作流，推送代码后自动编译 APK：

1. Fork 本仓库
2. 在 Settings → Secrets → Actions 中添加 `KEYSTORE_PASS`
3. 推送代码，Actions 自动编译
4. 在 Actions 页面的 Artifacts 中下载 APK
